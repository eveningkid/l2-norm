# l2 Norm
Implementation of the [l2 norm](https://en.wikipedia.org/wiki/Norm_(mathematics)#Euclidean_norm).  

Compute the euclidian/l2 norm of a vector.  

```bash
npm install l2-norm
yarn add l2-norm
```

```js
const l2norm = require('l2-norm');
l2norm([64, 42, 21]);
// ~79.38
```
